### EXAMPLE 2)

#### AC:
Sort the following array

```javascript
var arr = [0, 1, 5, 3, 4, 2, 9, 7, 8, 6];

var sortDesc = arr => {
    return arr.sort(
       (a, b) => b - a
    );

```
